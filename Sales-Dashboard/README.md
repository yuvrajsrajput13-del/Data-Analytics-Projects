# Sales Performance Dashboard

## Objective
Analyze sales data to track revenue, profit, and regional performance.

## Tools Used
- SQL
- Excel
- Power BI

## Key Insights
- Identified top-performing regions
- Analyzed monthly sales trends
- Found highest revenue-generating products

## KPIs
- Total Sales
- Total Profit
- Sales by Region
- Monthly Sales Trend

## Outcome
This dashboard helps stakeholders make data-driven decisions to improve sales strategy.
